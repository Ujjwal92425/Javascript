🔥 Mini Project for Practice (Event Bubbling, Capturing, Delegation based): 📋 Project Name: Task Manager with Delete + Complete Feature Requirements: Ek ul hogi jisme tasks honge. Ek input field aur button hoga naya task add karne ke liye. Har task ke aage: ✅ (Complete Button) ❌ (Delete Button) Pure ul pe hi event delegation lagana hai (individual task pe nahi). Delete Button dabane pe task list se hatana hai. Complete Button dabane pe task ka text green color me line-through hona chahiye. Bubbling / Capturing Concepts optional twist ke liye use kar sakte ho (bonus).
